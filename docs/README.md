# 🏠 État des lieux – PDF Automation System

## 📌 Overview

This system automatically generates a professional **PDF property inspection report** from a Google Form submission and sends it via email to the tenant.

## ⚙️ Features

* Google Forms → Google Sheets integration
* Automated PDF generation (HTML → PDF)
* Email delivery with attachment
* Clean and professional report

## 🚀 How it works

1. Tenant or agent fills the form
2. Data is stored in Google Sheets
3. n8n workflow is triggered
4. HTML template is generated
5. PDF is created automatically
6. Email is sent to the tenant

## 🛠️ Tech Stack

* n8n (automation)
* Google Forms / Sheets
* PDF API (PDF.co or similar)
* Gmail
